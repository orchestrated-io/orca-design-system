## Usage

The typography system requires you to import each subcomponent you want to use, for example:

```
import { H1, H2, H3, H4, H5, H6, P, Em, Strong, Code, Caption } from "orcs-design-system";
```

## Font families

By default, the design system uses **Open Sans**: https://fonts.google.com/specimen/Open+Sans with other sans serif families as fallback. The default font weight is 400, and the system also uses 600 for bold or emphasised text and 300 for light weights (to be used sparingly).

Font defaults can be changed via [systemtheme.js](https://github.com/orchestrated-io/orca-design-system/blob/master/packages/orcs-design-system/lib/systemtheme.js).

<Typeset fontSizes={systemtheme.fontSizes} fontFamily={systemtheme.font} />

## Text styling

### Size and emphasis

To render type smaller or larger, use `variant="small"` or `variant="large"`, or if needed (for example if you are already using a variant) you can use `fontSize` property as demonstrated below. Unless it is for a very specific purpose, strongly recommend minimising the resizing of text and simply using the appropriate type element the next size up (or down) on the design system scale.

<Preview>
<Story name="Styling/Sizing">
<>
<H5 variant="small">Small variant of H5</H5>
<H5 variant="large">Large variant of H5</H5>
<H5 fontSize={2}>Even maller H5, scaled down two steps in the fontSize array</H5>
<Text fontSize="10px">Very small text using a specific fontSize value</Text>
</>
</Story>
</Preview>

To render type with emphasis, use `variant="bold"` or wrap the type in a `<strong>` tag.

<Preview>
<Story name="Styling/Emphasis">
<>
<P variant="bold">Bold paragraph text lorem ipsum dolor sit amet</P>
<P>Only <strong>partially bolded</strong> paragraph</P>
</>
</Story>
</Preview>
