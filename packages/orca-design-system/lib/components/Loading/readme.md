As a general guide, use the large prop for when whole pages or sections are loading, and the default (small) loading for smaller elements.

```js
<Layout margin>
  <Loading />
</Layout>

<Layout margin>
  <Loading large />
</Layout>

<Layout margin>
  <Loading centered />
</Layout>

<Layout margin>
  <Loading large centered />
</Layout>

<Layout padding backgroundDark>
  <Loading inverted />
</Layout>
```
