This component supports a range of typographical sub components, as illustrated below.

Where possible try and use these rather than plain html tags like `<h1>`, `<small>`, `<p>` etc.

```js
  <Typography.H1 marginBottom>Heading 1</Typography.H1>

  <Typography.H1 marginBottom bold>Heading 1 bold</Typography.H1>

  <Typography.H2 marginBottom>Heading 2</Typography.H2>

  <Typography.H2 marginBottom bold>Heading 2 bold</Typography.H2>

  <Typography.H3 marginBottom>Heading 3</Typography.H3>

  <Typography.H3 marginBottom bold>Heading 3 bold</Typography.H3>

  <Typography.H4 marginBottom>Heading 4</Typography.H4>

  <Typography.H4 marginBottom bold>Heading 4 bold</Typography.H4>

  <Typography.H5 marginBottom>Heading 5</Typography.H5>

  <Typography.H5 marginBottom bold>Heading 5 bold</Typography.H5>

  <Typography.H6 marginBottom>Heading 6</Typography.H6>

  <Typography.H6 marginBottom bold>Heading 6 bold</Typography.H6>

  <Typography.P marginBottom>Paragraph text lorem ipsum dolor sit amet</Typography.P>

  <Typography.P marginBottom center>Centered paragraph text lorem ipsum dolor sit amet</Typography.P>

  <Typography.P marginBottom right>Right aligned paragraph text lorem ipsum dolor sit amet</Typography.P>

  <Typography.P marginBottom bold>Bold paragraph text lorem ipsum dolor sit amet</Typography.P>

  <Typography.P marginBottom bold grey>Makes text grey/muted colour</Typography.P>

  <Typography.P marginBottom bold success>Makes text success coloured</Typography.P>

  <Typography.P marginBottom bold warning>Makes text warning coloured</Typography.P>

  <Typography.P marginBottom bold danger>Makes text danger coloured</Typography.P>

  <Layout verticalMargin paddingHalf backgroundDark>

    <Typography.P bold white>Makes text white so it is visible on dark background.</Typography.P>

  </Layout>

  <Typography.Small marginBottom>Small text default</Typography.Small>

  <Typography.Small marginBottom bold>Small text bold</Typography.Small>

  <Typography.Small marginBottom grey>Small text muted</Typography.Small>

  <Typography.Small marginBottom uppercase>Small text uppercase</Typography.Small>

  <Typography.Large marginBottom>Large text</Typography.Large>

  <Typography.Large marginBottom bold>Large text bold</Typography.Large>

  <Typography.Quote>"Used to display block quotes lorem ipsum dolor sit amet" <em> - Adam, 2018</em></Typography.Quote>

  <Typography.Code>Used to display code snippets</Typography.Code>

```
